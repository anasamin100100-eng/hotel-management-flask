# Hotel Management System (Flask)

A web-based hotel management system built using Flask.

## Features
- User Registration & Login
- Room Search & Booking
- Dynamic Pricing with Tax
- Invoice Generation
- Admin Dashboard
- Payment Handling

## Tech Stack
- Python (Flask)
- SQLite
- HTML / Jinja2
- Bootstrap

## How to Run
```bash
pip install -r requirements.txt
python app.py
